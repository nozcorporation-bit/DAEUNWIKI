# raw/ — 원본 자료 (immutable)

## 무엇이 들어가나
핵심-맥락.md의 3단계 흐름에 필요한 모든 원자료:

- **경쟁사 키워드 데이터** — 최근 3개월 키워드 조회수 export (CSV/XLSX), 키워드 도구 스크린샷
- **광고 자료** — 라이브 중인 경쟁사 광고 영상 링크, 게시일·조회수·좋아요 메타데이터, 영상 스크린샷, 트랜스크립트
- **기사·리포트** — 마케팅 인사이트 클리핑, 업계 동향
- **첨부 이미지·파일**

## 절대 규칙
- **이 폴더의 파일은 수정·삭제하지 않습니다.** 원본의 무결성이 위키 신뢰성의 기반입니다.
- 같은 자료의 다른 버전이 들어오면 새 파일로 추가 (덮어쓰기 금지)
- 자료에 대한 요약·해석·분석은 여기 쓰지 말고 `wiki/`에 작성
- 출처 URL과 수집일은 파일 안에 명시 (또는 frontmatter에)

## 파일명 컨벤션
- 수집일을 앞에: `2026-05-09_브랜드명_광고리스트.md`
- 키워드 데이터: `2026-05-09_키워드데이터_3개월.csv`
- 광고 자료: `2026-05-09_브랜드명_광고제목.md`

## 권장 하위 구조 (필요할 때 LLM이 생성)
- `raw/keywords/` — 키워드 조회수 데이터
- `raw/ads/` — 광고 영상 링크·메타데이터·트랜스크립트
- `raw/articles/` — 기사·리포트 클리핑
- `raw/assets/` — 이미지·스크린샷·첨부

## LLM이 raw/에서 할 수 있는 유일한 일
- **읽기**
- 새 파일을 사용자 동의하에 적절한 하위 폴더로 정리
- 파일 안에 frontmatter 추가 (`source_url`, `collected_at`, `notes`) — 단, 본문 내용은 손대지 않음
