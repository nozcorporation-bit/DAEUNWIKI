# 핵심 맥락

이 위키는 **1인 창업가의 AI 세컨드 브레인**입니다. **코르셋 브랜드 런칭 준비 중**이며, 영감을 주는 레퍼런스 광고·콘텐츠에서 훅·설득구조를 추출해 **자신의 런칭 콘텐츠·광고 소재 설계에 직접 활용**하는 것이 목표입니다.

---

## 도메인

- **카테고리**: 코르셋 (이너웨어·바디웨어·셰입웨어)
- **제품 단계**: 런칭 준비 중 (아직 출시 전)
- **사용자 역할**: 본인이 직접 브랜드 콘텐츠·광고 카피·크리에이티브 제작 → 위키의 "적용 도출안"은 곧바로 자신의 런칭 콘텐츠로 투입

## 자료의 성격 — "레퍼런스" 중심

이 위키의 자료는 꼭 **경쟁사**일 필요 없음. **영감을 주는 모든 콘텐츠**가 대상:
- 다른 카테고리(뷰티·패션·셀프케어·여성건강) 브랜드의 광고
- 인플루언서·크리에이터의 오가닉 콘텐츠 (Instagram Reels, TikTok, YouTube Shorts)
- 동종 카테고리(코르셋·셰입웨어) 직접 경쟁사 광고도 포함되지만 메인은 아님
- 시각·메시지·후킹 어느 측면에서든 "이거 좋다" 싶으면 ingest

## 데이터 소스

- **레퍼런스 콘텐츠** → `raw/refs/` (메인)
  - `raw/refs/instagram/` — Reels, 피드 광고·오가닉
  - `raw/refs/tiktok/`
  - `raw/refs/youtube/` — Shorts, 일반 영상
- **광고 라이브러리** → `raw/ads/` (서브, 런칭 후 우선순위↑)
  - `raw/ads/meta/` — Facebook·Instagram 광고 라이브러리에서 의도적으로 찾은 광고
  - `raw/ads/tiktok/` — TikTok Creative Center
- **키워드 데이터** → `raw/keywords/` (런칭 후 더 중요)
  - 네이버 키워드도구·블랙키위·셀러스프라이트
- 각 자료에 **링크 + 게시자 + 게시일 + engagement 지표**(좋아요·댓글·조회수) 필수

---

## 작업 흐름

### 1) 레퍼런스 ingest (현재 메인)
새 레퍼런스가 들어오면 5개 항목 추출:
- **훅 (hook)** — 첫 1~3초의 시선 캡처 장치
- **후킹포인트** — 시청 지속을 만드는 결정적 순간
- **설득구조** — 문제 제기 → 솔루션 → 증거 → CTA 같은 논리 흐름
- **전환 논리** — 행동 결정의 결정타 (할인 / 후기 / 권위 / 결핍감 / 신뢰지표 등)
- **내 코르셋 브랜드 적용 도출안** ← **가장 중요**

추가:
- 게시자(인플루언서·브랜드)는 `wiki/entities/`에 entity로 등록 → 같은 사람의 후속 콘텐츠를 묶을 수 있게
- 반복 발견되는 패턴(예: "비포·애프터 비주얼", "권위 빌드업")은 `wiki/concepts/`에 누적

### 2) 키워드 추적 (런칭 후 우선순위↑)
런칭이 가까워지면 코르셋·셰입웨어 관련 키워드 모니터링 시작.

**급상승 정의 (작업 기준 — 운영하면서 조정):**
- 메인: 전월 대비 +150% **AND** 최근 월 조회수 1,000회 이상
- 보조: 절대 증가량 +1,000회 이상

### 3) 적용 도출안 → 실제 콘텐츠
도출안은 **카피 초안 또는 비주얼 컨셉 수준**이어야 함. 추상적 인사이트 금지. 그대로 런칭 소재 작업에 들어갈 수 있게.

---

## 이 위키가 잘 작동한다는 신호

- 새 콘텐츠 만들 때 "비슷한 톤의 그 레퍼런스 뭐였지?" → 위키에서 **5초 안에** 회수
- 같은 인플루언서·같은 패턴을 두 번 분석하지 않는다
- 도출안이 실제 런칭 콘텐츠로 채택된 비율 누적 → `#채택` 태그
- 런칭 시점에 "어떤 메시지·훅으로 갈지" 결정이 위키 안에 수렴되어 있음
