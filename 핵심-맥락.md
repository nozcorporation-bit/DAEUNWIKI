# 핵심 맥락

이 위키는 **1인 마케터의 AI 세컨드 브레인**입니다. 이미 판매 중인 식품·건강기능식품 제품의 광고 효율을 개선하기 위해, 경쟁사의 떠오르는 키워드와 광고를 추적하고 **곧바로 광고 카피·크리에이티브 소재로 변환**하는 것이 목표입니다.

---

## 도메인

- **카테고리**: 식품·건강기능식품
- **제품 단계**: 이미 판매 중, 광고 운영·최적화 단계 (신규 런칭 아님)
- **사용자 역할**: 본인이 직접 광고 카피·크리에이티브 제작 → 위키의 "적용 도출안"은 외주 브리프·보고서가 아니라 **곧바로 소재 작업에 투입되는 카피 초안·비주얼 컨셉** 수준이어야 함

## 데이터 소스

- **키워드 데이터**: 네이버 키워드도구 / 블랙키위 / 셀러스프라이트 → `raw/keywords/`에 export로 저장
- **광고 자료**: 플랫폼별로 분리해서 저장
  - 메타 광고 라이브러리 (Facebook·Instagram) → `raw/ads/meta/`
  - TikTok Creative Center → `raw/ads/tiktok/`
  - YouTube → `raw/ads/youtube/`
- 각 광고 자료에는 **영상 링크 + 광고주 + 게시일 + 조회수 + engagement 지표** 필수
  - 플랫폼별 메타데이터가 다름: TikTok(좋아요·공유), Meta(활성기간·노출 플랫폼), YouTube(조회수·좋아요·댓글)

---

## 작업 흐름 (3단계)

### 1) 급상승 키워드 추출
최근 **3개월** 경쟁사 키워드 데이터에서 최근 **1개월** 조회수가 급격히 오른 키워드를 추출.

**급상승 정의 (작업 기준 — 운영하면서 조정):**
- **메인 트리거**: 전월 대비 조회수 +200% 이상 **AND** 최근 월 조회수 1,000회 이상 (저볼륨 노이즈 컷)
- **보조 트리거**: 절대 증가량 +1,000회 이상 (대형 키워드의 가속 포착)
- 둘 중 하나라도 만족 → "급상승"
- 식품·건강기능식품은 **계절성·건강 트렌드**(면역, 다이어트, 수면, 신규 원료 등)에 민감 → 단발성 스파이크인지 추세인지 3개월 곡선을 함께 보고 `wiki/keywords/`에 분류 태그(`#스파이크` / `#추세`)

### 2) 경쟁사 광고 영상 정리
1번에서 뽑힌 키워드로 검색 → 라이브 중인 경쟁사 광고를 수집:
- 영상 링크, 광고주, 게시일, 조회수, 좋아요/engagement
- 광고 1건당 `wiki/ads/` 페이지 1개

### 3) 광고 분석 → 적용 도출안
각 광고에 대해 **5개 항목 빠짐없이**:
- **훅 (hook)** — 첫 1~3초의 시선 캡처 장치
- **후킹포인트** — 시청 지속을 만드는 결정적 순간
- **설득구조** — 문제 제기 → 솔루션 → 증거 → CTA 같은 논리 흐름
- **전환 논리** — 구매 결정의 결정타 (할인 / 후기 / 권위 / 결핍감 / 신뢰지표 등)
- **우리 제품 적용 도출안** ← **가장 중요**
  - 추상적 인사이트 금지
  - **카피 초안 또는 비주얼 컨셉 수준**으로, 그대로 소재 작업에 들어갈 수 있게

---

## 이 위키가 잘 작동한다는 신호

- 새 광고 소재를 만들 때 "지난번 분석한 그 광고가 뭐였지?" → 위키에서 **5초 안에** 찾는다
- 같은 키워드·광고 패턴을 **두 번 분석하지 않는다** (이미 위키에 있음을 LLM이 자동 인지)
- 적용 도출안이 실제 광고 소재로 채택된 비율이 누적된다 → 채택된 도출안에는 `#채택` 태그
